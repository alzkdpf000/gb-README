




# 📊 공공데이터분석(기획배경)
최근 언론 보도에 따르면 해외여행을 필수로 여기는 Z세대가 빠르게 증가하고 있으며, 실제 통계 데이터를 기반으로 해외여행 수요가 늘어나고 있는지를 검증하고자 시각화 분석을 진행하였다. 그 결과, 전반적인 해외여행 증가 추세가 명확하게 확인되었습니다.<img width="892" height="497" alt="Image" src="https://github.com/user-attachments/assets/2e785061-087a-4c09-8021-0be51391ba9b" />


## 1. 해외여행 수요의 장기적 변화
> <img width="558" height="464" alt="스크린샷 2025-11-20 오후 3 49 59" src="https://github.com/user-attachments/assets/c955457e-bd72-41b7-bc74-bc1dc46d3d00" />

시각화된 자료를 통해 연도별 해외여행객 수를 비교한 결과, 장기적으로 꾸준한 상승 흐름을 보였습니다. 다만, 특정 시기에는 세계적·국가적 사건으로 인해 일시적인 감소가 나타났습니다.
### 주요 감소 요인
- **1998년 IMF 외환위기**
경제 위기로 국민 소득과 소비 여력이 감소하며 해외여행 수요가 크게 위축되었습니다.
- **2009년 신종 인플루엔자(신종플루) 유행**
전염병 확산 우려로 해외 이동이 감소했습니다.
- **2020–2022년 코로나19 팬데믹**
역사적 수준의 국경 봉쇄와 항공편 축소로 해외여행객 수가 급격히 하락하였습니다.
## 2. 코로나 이후 회복세
특히 2023년 이후 여행 수요가 급격히 회복되었으며, 코로나 이전을 넘어서는 성장세가 관측되고 있습니다. 이는 다음과 같은 요인과 관련이 있다고 생각합니다.
억눌렸던 여행 욕구의 폭발적 증가(보복여행·욜로 소비)
저비용 항공사(LCC)의 확장
SNS 활용 증가로 인한 여행 정보 접근성 확대
MZ·Z세대 중심의 경험 소비 트렌드 강화
## 3. 결론
종합적으로 살펴보면, 해외여행 수요는 단기적인 외부 충격으로 일시적 감소를 겪었으나 장기적으로 지속적인 증가 추세를 보이고 있습니다.
특히 최근 Z세대를 중심으로 해외여행을 **필수적 경험**으로 여기는 인식이 확산되면서 향후에도 여행 수요는 지속적으로 확대될 것으로 예상됩니다.

## 🧭 총평
### 기획
에러 처리의 경우, 게시글이 존재하지 않거나 회원 정보가 유효하지 않은 경우 등 다양한 예외 상황에 대해 미리 논의하고 기획한 점이 매우 유익했습니다. 서버 측에서는 이러한 사전 기획 덕분에 예외 발생 시의 흐름(예: 에러 페이지 이동, 토스트 메시지 처리 등)을 명확히 설계할 수 있었고, 실제 개발 단계에서도 혼선을 줄일 수 있었습니다. 이 경험을 통해 서버 로직에서도 ‘미리 기획된 시나리오’가 얼마나 중요한지 체감했습니다.
### 협업
이번 프로젝트에서 새로 배운 Spring Security와 JWT를 적용하는 과정에서, URL 접근 권한 설정이나 토큰 유무에 따른 처리 로직 등 핵심적인 보안 설정 정보를 팀원 간에 즉시 공유하지 못하는 상황이 있었습니다. 이를 계기로, 중요한 설정이나 로직 변경이 있을 때는 짧게라도 회의를 진행하고, Slack 같은 협업 도구를 더욱 적극적으로 활용해야 한다는 필요성을 느꼈습니다. 이전보다 기술 수준은 높아졌지만, 정보 공유 속도와 방법 또한 그에 맞게 개선되어야 함을 깨달았습니다.
### 좋았던 점
이전 프로젝트보다 훨씬 다양한 기술을 경험해 볼 수 있어서 좋았습니다. AWS S3, AWS IAM, Spring Security, AOP, Swagger, JWT 등 새로운 기술들을 직접 적용하며 배운 점이 많았고, 실무 감각을 키울 수 있는 좋은 기회였습니다. 특히 배포 및 인증 과정을 직접 다뤄봄으로써 백엔드 전반에 대한 이해도가 크게 향상되었습니다.
### 아쉬웠던 점
